# cinema-web-directory-archive
本仓库用于整理影视网站的基础目录信息，对公开网络环境中的影视类站点进行统一归档。记录内容侧重站点层面的客观信息，不包含任何影视资源或播放入口。所有整理行为基于公开资料完成，仓库不对站点运营情况作出评价，仅作为信息索引与资料留存用途，不涉及任何形式的引导或背书。
